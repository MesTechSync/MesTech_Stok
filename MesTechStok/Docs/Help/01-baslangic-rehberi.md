# Baslangic Rehberi

MesTech'e hosgeldiniz. Bu rehber ilk kurulum adimlarini anlatir.

## 1. Hesap Olusturma
- panel.mestech.tr adresine gidin
- "Kayit Ol" butonuna tiklayin
- Firma bilgilerinizi girin (Firma Adi, Vergi No, Telefon, E-posta)
- Sifrenizi belirleyin (en az 8 karakter, buyuk harf + rakam)

## 2. Ilk Giris
- E-posta ve sifrenizle giris yapin
- Onboarding sihirbazi otomatik baslar (7 adim)

## 3. Onboarding Adimlari
1. **Kayit** — Tamamlandi (hesabiniz olusturuldu)
2. **Firma Bilgileri** — Vergi dairesi, adres, IBAN bilgilerini girin
3. **Ilk Magaza** — Hangi pazaryerinden baslamak istiyorsunuz?
4. **Platform Baglantisi** — Pazaryeri secin (Trendyol, Hepsiburada, vb.)
5. **Credential Girisi** — API Key ve Secret bilgilerini girin
6. **Ilk Senkronizasyon** — Urunleriniz otomatik cekilir
7. **Dashboard** — Hazirsiniz! Ana sayfaya yonlendirilirsiniz.

## Sikca Sorulan Sorular
- **API Key nedir?** Pazaryerinin size verdigi erisim anahtaridir. Magaza yonetim panelinden alinir.
- **Kac magaza baglanabilir?** Planinizdaki limite gore. Baslangic: 1, Profesyonel: 5.
- **Verilerim guvenli mi?** Tum veriler AES-256 ile sifrelenir, KVKK uyumludur.
