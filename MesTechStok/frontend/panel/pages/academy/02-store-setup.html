Magaza Ekleme

~8 dakika
Adim 1: Yeni Magaza Ekle

Sol menuden Ayarlar > Magazalar sayfasina gidin. "Yeni Magaza Ekle" butonuna tiklayin. Karsiniza desteklenen pazaryerlerinin listesi gelecektir: Trendyol, Hepsiburada, Ciceksepeti, N11, Pazarama, Amazon, eBay ve daha fazlasi.

Adim 2: API Bilgilerini Girin

Sectiginiz pazaryerinin satici panelinizden API anahtari (API Key) ve gizli anahtar (Secret Key) bilgilerini alin. Bu bilgileri MesTech'teki ilgili alanlara yapisitirin. Her pazaryerinin API anahtar alma sureci farkli olabilir.

Uyari: API anahtarlarinizi asla baskasıyla paylasmayın. Bu anahtarlar magazaniza tam erisim saglar.
Adim 3: Baglanti Testi

API bilgilerini girdikten sonra "Baglanti Test Et" butonuna tiklayin. Sistem pazaryeri ile iletisim kuracak ve baglantiyi dogrulayacaktir. Basarili ise yesil onay mesaji goreceksiniz.

Ipucu: Baglanti basarisiz olursa API anahtarlarini kontrol edin. Bazi pazaryerleri IP adresi kisitlamasi uygulayabilir.
Adim 4: Magaza Ayarlarini Yapilandirin

Baglanti basarili olduktan sonra magazaniz icin ek ayarlar yapin: varsayilan KDV orani, kargo sablonu, komisyon hesaplama yontemi ve stok senkronizasyon sikligi. Bu ayarlar daha sonra da degistirilebilir.

Adim 5: Ilk Senkronizasyonu Baslatin

Magazaniz eklendikten sonra ilk senkronizasyon otomatik baslar. Bu islem mevcut urunlerinizi, siparislerinizi ve stok bilgilerinizi ceker. Urun sayisina bagli olarak bu islem birkac dakika surebilir.

Ipucu: Birden fazla magaza ekleyebilirsiniz. Her magaza bagimsiz olarak senkronize edilir.
