12

Sistem Ayarlari

~10 dakika
Ders Hedefleri
  • API anahtari olusturma ve yonetmeyi ogreneceksiniz
  • Bildirim kanallarini yapilandiracaksiniz
  • Yedekleme ve geri yukleme islemlerini kavrayacaksiniz
  • Sistem sagligi izleme araclarini kullanacaksiniz
Adim 1: API Anahtarlari

Ayarlar > API sayfasindan API anahtari olusturabilir ve yonetebilirsiniz. API anahtarlari dis sistemler ile entegrasyon icin kullanilir.

  1. "Yeni API Anahtari" butonuna tiklayin
  2. Anahtar icin aciklayici bir isim girin
  3. Yetkilerini secin (hangi modullere erisebilir)
  4. Son kullanma tarihini belirleyin
Dikkat: API anahtari olusturuldugunda sadece bir kez goruntulenir. Anahtari guvenli bir yere kaydedin. Kaybederseniz yeni anahtar olusturmaniz gerekir.
Adim 2: Bildirim Ayarlari

Ayarlar > Bildirimler sayfasindan bildirim kanallarini ve tetikleyicileri yapilandirin.

  • E-posta bildirimleri: SMTP ayarlari veya entegre e-posta servisi
  • Sistem ici bildirimler: panel icinde pop-up uyarilar
  • Webhook bildirimleri: Slack, Teams veya ozel URL'ye gonderim
Ipucu: Kritik uyarilari (stok bitti, siparis gecikti) icin e-posta + sistem ici birlesik bildirim kurun. Daha az onemli bilgileri sadece sistem icinde gosterin.
Adim 3: Yedekleme ve Geri Yukleme

MesTech otomatik gunluk yedekleme yapar. Manuel yedek almak veya geri yuklemek icin Ayarlar > Yedekleme sayfasini kullanin.

  • Otomatik yedekleme: her gece saat 03:00'te alinir
  • Yedekler MinIO nesne depolamada 30 gun saklanir
  • Manuel yedek almak icin "Simdi Yedekle" butonuna tiklayin
  • Geri yukleme icin yedek listesinden tarihi secin ve "Geri Yukle" tiklayin
Uyari: Geri yukleme mevcut verilerin uzerine yazar. Bu islemi yapmadan once guncel yedek almaniz kesinlikle onerilir.
Adim 4: Sistem Sagligi Izleme

MesTech Prometheus + Grafana monitoring altyapisi ile sistem sagligini izler.

  • /health endpoint'i ile servis durumunu kontrol edin
  • Grafana dashboard'una port 3000'den erisin
  • Seq loglarina port 8080'den erisin — hata takibi
  • Docker container durumlarini izleyin
