04

Pazaryeri Entegrasyonu

~15 dakika
Ders Hedefleri
  • Desteklenen pazaryerlerini ve ozelliklerini taniyacaksiniz
  • API anahtari ile pazaryeri baglantisi kurabileceksiniz
  • Urun ve stok senkronizasyonunu yapilandiracaksiniz
  • Platform bazli fiyatlandirma stratejisi olusturabileceksiniz
Adim 1: Desteklenen Pazaryerleri

MesTech 12 farkli pazaryeri ile entegrasyon saglar. Her platformun kendine ozel API yapisi ve is kurallari vardir.

  • Turkiye: Trendyol, Hepsiburada, N11, Ciceksepeti, Pazarama, PttAVM
  • Uluslararasi: Amazon, eBay, Ozon
  • E-ticaret altyapisi: Shopify, WooCommerce, OpenCart
Adim 2: API Baglantisi Kurma

Her pazaryeri icin API anahtari almaniz gerekir. Bu anahtar pazaryerinin satici panelinden temin edilir.

  1. Pazaryerinin satici paneline giris yapin
  2. API / Entegrasyon bolumunden API anahtari olusturun
  3. MesTech Panel > Ayarlar > Pazaryerleri sayfasina gidin
  4. Ilgili platformu secin ve API bilgilerini girin
  5. "Baglanti Testi" butonuyla dogrulamayi yapin
Dikkat: API anahtarlarini kimseyle paylasmayiniz. Anahtarlar BCrypt ile sifrelenerek saklanir ancak guvenligi sizin sorumlulugunuzdadir.
Adim 3: Urun Senkronizasyonu

Baglanti kurulduktan sonra urunlerinizi pazaryerlerine gonderebilirsiniz. Senkronizasyon ayarlari platforma ozeldir.

  • Otomatik senkronizasyon: Urun degisikliklerini anlik yansitir
  • Manuel senkronizasyon: Secilen urunleri toplu olarak gonderir
  • Senkronizasyon loglarini "Entegrasyon > Loglar" sayfasindan takip edin
Ipucu: Ilk senkronizasyonda az sayida urunle test yapin. Tum urunleri gondermeden once kategori eslestirmelerinin dogru oldugundan emin olun.
Adim 4: Platform Bazli Fiyatlandirma

Her platform icin farkli fiyat belirleyebilirsiniz. Komisyon oranlari platforma gore degisir ve MesTech bu oranlari otomatik cekmektedir.

  1. Urun detayinda "Platform Fiyatlari" sekmesine gidin
  2. Her platform icin satis fiyatini ayri belirleyin
  3. Komisyon orani otomatik hesaplanir ve net kar gosterilir
Uyari: Pazaryerlerinin komisyon oranlari kategoriye gore degisir. Daha az komisyon icin dogru kategori eslestirmesi kritik onem tasir.
