Ileri Duzey Ipuclari

~15 dakika
Adim 1: Klavye Kisayollari

MesTech'te hiz kazanmaniz icin klavye kisayollari kullanabilirsiniz. Ctrl+K ile hizli arama acin, Ctrl+N ile yeni urun ekleyin, Ctrl+S ile degisiklikleri kaydedin. Tam kisayol listesine Ayarlar > Kisayollar sayfasindan ulasabilirsiniz.

Ipucu: Ctrl+K ile acilan hizli arama tum sayfalarda calisir. Urun, siparis veya musteri arayabilirsiniz.
Adim 2: Toplu Islem Stratejileri

Toplu fiyat guncelleme ile sectiginiz urunlerin fiyatini yuzde veya sabit tutar olarak degistirin. Toplu stok guncelleme ile Excel dosyasindan stok miktarlarini aktarin. Toplu kategori degistirme ile urunleri farkli kategorilere tasiyin.

Adim 3: API Kullanimi

MesTech REST API ile kendi otomasyon senaryolarinizi olusturabilirsiniz. Ayarlar > API sayfasindan API anahtarinizi olusturun. Swagger dokumantasyonundan tum endpoint'leri inceleyebilirsiniz. JWT token ile guvenli erisim saglanir.

Uyari: API anahtarinizi guvenli saklayin. Yetkisiz erisim tespit edildiginde anahtar otomatik iptal edilir.
Adim 4: Performans Ipuclari

Buyuk urun kataloglarina sahipseniz sayfa yukleme surelerini optimize edin: sayfalama kullanin (sayfa basina 50-100 urun), gereksiz sutunlari gizleyin ve filtre sonuclarini kaydedin. Tarayici onbellegini periyodik olarak temizleyin.

Adim 5: Zamanlanmis Gorevler

Hangfire tabanli gorev zamanlayici ile otomatik islemler tanimlayin: her gece stok senkronizasyonu, haftalik rapor gonderimi, gunluk yedekleme. Ayarlar > Zamanlanmis Gorevler sayfasindan yonetebilirsiniz.

Ipucu: Yogun saatlerde zamanlanmis gorev calistirmaktan kacinin. Gece 02:00-05:00 arasi ideal zamanlama dilimidir.
Adim 6: Destek ve Kaynak

Sorulariniz icin SSS sayfasini inceleyin, destek@mestech.tr adresine e-posta gnderin veya canli destek ile iletisime gecin. MesAkademi'deki dersleri tekrar izleyerek bilginizi tazeleyebilirsiniz. Yeni ozellikler icin Degisiklik Gunlugu sayfasini takip edin.
